# Kbd

Renders a keyboard key indicator.

## Import

```tsx
import { Kbd } from '@florex/ui';
```

## Usage

```tsx
<Kbd>⌘</Kbd>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | `ReactNode` | — | Key label |
| className | `string` | — | Additional CSS classes |

## Examples

### Keyboard shortcut

```tsx
<p>Press <Kbd>⌘</Kbd> + <Kbd>K</Kbd> to search.</p>
```
