# Table

A composable data table with header, body, rows, and cells.

## Import

```tsx
import { Table, TableHeader, TableBody, TableRow, TableHead, TableCell } from '@florex/ui';
```

## Usage

```tsx
<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Name</TableHead>
      <TableHead>Role</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Ana García</TableCell>
      <TableCell>Admin</TableCell>
    </TableRow>
  </TableBody>
</Table>
```
