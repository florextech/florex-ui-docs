# Divider

A horizontal rule, optionally with a centered label.

## Import

```tsx
import { Divider } from '@florex/ui';
```

## Usage

```tsx
<Divider />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| label | `string` | — | Centered text label |
| className | `string` | — | Additional CSS classes |

## Examples

### With label

```tsx
<Divider label="or" />
```

### Plain

```tsx
<Divider />
```
