# IconButton

Used to render an icon-only button.

## Import

```tsx
import { IconButton } from "@florexlabs/ui";
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `icon` | `ReactNode` | — | Icon element |
| `aria-label` | `string` | **required** | Accessible label |
| `variant` | `"primary" \| "secondary" \| "ghost"` | `"ghost"` | Visual style |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Button size |
