# CodeBlock

A multi-line code block with optional title and copy-to-clipboard button.

## Import

```tsx
import { CodeBlock } from '@florex/ui';
```

## Usage

```tsx
<CodeBlock>{`const greeting = "Hello, world!";
console.log(greeting);`}</CodeBlock>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | `string` | — | Code content (must be a string) |
| title | `string` | — | Optional header label (e.g. filename) |
| className | `string` | — | Additional CSS classes |
