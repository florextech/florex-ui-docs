# Status

A status indicator with a colored dot and label text.

## Import

```tsx
import { Status } from '@florex/ui';
```

## Usage

```tsx
<Status value="success">Online</Status>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `'success' \| 'warning' \| 'danger' \| 'neutral'` | — | Status type (controls dot color and glow) |
| children | `ReactNode` | — | Status label |
| className | `string` | — | Additional CSS classes |

## Examples

### All statuses
