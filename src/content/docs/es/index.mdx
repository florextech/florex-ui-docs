## Instalación

```bash
pnpm add @florexlabs/ui
pnpm add react react-dom tailwindcss
pnpm add @phosphor-icons/react  # opcional
```

## Configuración

```css
@import "tailwindcss";
@import "@florexlabs/ui/florex.css";
```

## Uso rápido

```tsx
import { Button, Card, Badge } from "@florexlabs/ui";

<Card variant="hover">
  <Button variant="primary">Guardar</Button>
  <Badge tone="brand">Nuevo</Badge>
</Card>
```
